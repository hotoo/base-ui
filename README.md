# base-ui

支付宝移动端样式解决方案
___

## Features

* 更简洁的 Dom 结构
* 更少的图片
* 更易扩展

## Browsers

* iOS 4.0+
* Android 2.2+
* Window Phone 8+

## How to use

1. install node
2. install nico for doc
3. npm install
4. Makefile:
  * `make build` 生成编译之后的样式
  * `make doc` 生成文档
  * `make watch` 开启本地服务，用来预览文档，调试。
